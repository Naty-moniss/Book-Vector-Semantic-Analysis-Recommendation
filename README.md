📝 Description
Ce projet est né de la volonté d'unir trois piliers de mon parcours académique : l'application pratique du Big Data, l'étude de l'Algèbre Linéaire et ma passion pour la littérature. L'objectif est de transformer des descriptions textuelles de livres en vecteurs mathématiques pour identifier des modèles de similarité.

🎯 Objectifs Académiques
Intégration Data Science : Appliquer des concepts de Big Data et d'Analytics.

Préparation L2/L3 Informatique : Renforcer mes bases en mathématiques appliquées (SVD, matrices) pour mes futures candidatures en France.

Analyse Sémantique : Utiliser le Traitement du Langage Naturel (NLP) pour organiser des données complexes.

🧪 Concepts Mathématiques Utilisés
TF-IDF : Vectorisation du texte.

Similarité Cosinus : Calcul de la proximité entre les œuvres.

SVD (Singular Value Decomposition) : Réduction de dimensionnalité pour la visualisation 2D.

🛠️ Outils
Python (Pandas, Scikit-Learn, Plotly).



Português:


📝 Descrição
Este projeto une a aplicação prática de Big Data e Analytics com o estudo de Álgebra Linear e Literatura. O sistema transforma descrições de livros em representações matemáticas para identificar padrões de similaridade e agrupamentos semânticos.

🎯 Objetivos
Prática de Analytics: Aplicar técnicas de análise de dados em larga escala.

Base para Universidades Francesas: Fortalecer o entendimento de álgebra linear aplicada, essencial para o ingresso em Licence Informatique na França.

Visualização Interativa: Criar uma interface onde o usuário explora a "distância" matemática entre diferentes autores e obras.

🧪 Conceitos Aplicados
TF-IDF: Transformação de palavras em coordenadas numéricas.

Similaridade de Cosseno: Medição do ângulo entre vetores para definir semelhança.

SVD: Redução de complexidade para visualização de dados.

🛠️ Ferramentas
Python (Pandas, Scikit-Learn, Plotly).

🚀 Installation / Instalação
Bash
pip install pandas scikit-learn plotly
python main.py
